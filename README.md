PW生成 (パスワード生成アプリ)
指定した条件（文字数や記号の有無）に合わせて、安全なランダムパスワードを自動生成するデスクトップアプリです。

1.特徴
  ・直感的なGUI: CutomTkinterを採用したモダンなデザイン。
  ・簡単コピー: 生成したパスワードをワンクリックでクリップボードにコピー可能。
  ・カスタマイズ: パスワードの長さや使用する文字種（英数字・記号）を自由に選択。

2.実行方法
　①前準備
　　Python 3.x がインストールされていることを確認してください。
　②ライブラリのインストール
　　ターミナル（PowerShellなど）で以下のコマンドを実行し、必要なライブラリをインストールします。 pip install -r requirements.txt
　③アプリの起動
　　メインプログラムを実行します。 

　　python password_gen.py

3.開発環境
  OS: Windows 11
  言語: Python 3.12+
  主要ライブラリ: CustomTkinter, pyperclip, Pillow
